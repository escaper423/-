  데이터베이스 텀프로젝트 시 작성한 소스들입니다.

3인으로 수행하였으며, 수강신청시스템에 관한 데이터베이스를 작성하고, 간단한 홈페이지를 제작,

이를 관리하는 방식으로 작성하였습니다.

본인이 작성한 항목들은 다음과 같습니다.

학생 기능
  -> 수강 확인, 취소 ok
  -> 수강 편람을 확인 -> 수강 신청 ok
  -> 시간표 보기 (개인, 교수별, 강의실별을 실제 시간표 보는 것과 동일하게) ok
  -> 성적 확인(학기별, 전체) ok

관리자 기능
  -> 교수 및 조교의 명단 입력.ok
  -> 각 건물의 강의실이 비는 시간 조회 ok

보다 자세한 사항은 폴더 내의 계획서에서 확인하실 수 있습니다.
